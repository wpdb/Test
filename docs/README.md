# AI Documentation

This folder contains custom instructions for the AI assistant.

## Files

| File | Purpose |
|------|---------|
| `ai-instructions.md` | Your preferences, custom APIs, guidelines |

## How It Works

- **ai-instructions.md** - Edit this to tell the AI your preferences (coding style, design choices) and document any custom utilities you've added.

The AI reads meaningful content from `ai-instructions.md` automatically with each message. Blank template text and comments are ignored.
